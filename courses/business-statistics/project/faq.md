1. How should one approach the E-news Express project?

- Before starting the project, please read the problem statement carefully and go through the criteria and descriptions mentioned in the rubric.
- Once you understand the task, download and import the dataset into a jupyter notebook to get started with the project.
- To work on the project, you should first start with an exploratory data analysis followed by descriptive statistics and visualization to understand the data. 
- Once the EDA is completed, you can use the data to start answering the business problem that we are trying to address. It is important to close the analysis with key findings and recommendations to the business.
 

2. Is it necessary to get a random sample from the dataset to perform the tests? 

No, you should assume that the dataset is a simple random sample from the population, and the sample observations are independent of each other.

 
3. Does this project require the same level of extensive visualizations and EDA as the previous project?

You are expected to perform basic dataset exploration, univariate analysis, bivariate analysis, missing value check, etc.

However, please be sure to provide the observations/insights from your analysis as given in the previous project.

 
4. What kind of visuals are expected to be performed to describe the hypothesis testing questions?

You are expected to use visual analysis for comparison of the data that will be used in hypothesis testing. 
For example, `boxplot()` can be used to compare the means in an ANOVA test. 
Similarly, you can draw any suitable plot to find visual evidence against your formulated hypotheses. 
You can use boxplot, barplot, or any other suitable plot depending upon the type of data used in the test.

 
5.  How to calculate the conversion rate?

Conversion rate is the proportion of users who visit the landing page and get converted. 
For example, if out of 50 users, 20 gets converted. Then the conversion rate is `20 / 50 = 0.4`.

 
6. Is it necessary to follow both the rejection region Approach and p_value approach for the Hypothesis testing?

It is not necessary to follow both the rejection region approach and p-value approach for the hypothesis testing. 
You can use either of them (preferably the p-value approach) to get the inference. 
The rejection region approach might be a bit lengthy and can take more time as compared to the p-value approach.

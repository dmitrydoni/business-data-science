## Rubric

### Exploratory Data Analysis

- Problem definition
- Univariate analysis
- Bivariate analysis
- Use appropriate visualizations to identify the patterns and insights
- Answers to EDA questions provided
- Key meaningful observations on individual variables and the relationship between variables

### Data Preprocessing

- Missing Value Treatment (with rationale if needed)
- Outlier Detection and Treatment (with rationale if needed)
- Feature Engineering (with rationale if needed)
- Data Scaling (with rationale if needed)
- Train-test split

### Model Building

- Choose the metric to optimize for the problem
- Build the following models
    - KNN Classifier
    - Naive-Bayes Classifier
    - SVM Classifier
- Check and comment on model performance across different metrics

### Model Performance Improvement

- Tune the following models to improve performance
    - KNN Classifier
    - Naive bayes
    - SVM Classifier (at least 5 combinations of parameters)
- Check and comment on tuned model performance across different metrics

### Actionable Insights & Recommendations

- Compare the performances of all the models built
- Choose the final model with a proper rationale
- Write down insights from the analysis conducted
- Provide actionable business recommendations

### Notebook Overall Quality

- Structure and flow
- Well commented code
- All code executed and necessary output visible
- No Errors

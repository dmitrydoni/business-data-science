## Overview

Coursework for a Probability Theory and Mathematical Statistics course (GeekBrains, supervisor: Yury Lytkin, author: Dmitry Doni). 
The repo contains Jupyter notebooks with solved exercises progressing from basic probability through regression and ANOVA, plus a capstone project.

## Key conventions

- Notebooks are in Russian (task descriptions) with English code comments and variable names
- External datasets are referenced at `../../datasets/` (relative path, outside this repo)
- Libraries used: `numpy`, `pandas`, `scipy.stats`, `matplotlib`, `seaborn`
- Many formulas are implemented manually first (from scratch using numpy), then verified with library functions — preserve this pattern when adding solutions
- Notebooks use `np.math.factorial` for combinatorics and `scipy.stats` for distributions/hypothesis testing
